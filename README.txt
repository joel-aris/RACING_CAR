Description du Projet

Ce projet est un jeu de course de voiture développé en HTML, CSS, JavaScript et PHP. Le but du jeu est de contrôler une voiture et d'éviter les obstacles qui apparaissent sur la route. Les scores sont enregistrés dans une base de données MySQL pour permettre aux joueurs de comparer leurs performances.

Le projet a été créé dans le cadre de mon auto-formation en sciences informatiques, en particulier pour explorer les concepts de développement web et de programmation JavaScript/PHP.
Fonctionnalités

    🚗 Contrôle de la voiture : Utilise les touches fléchées gauche et droite pour déplacer la voiture entre les trois voies.

    🛑 Éviter les obstacles : Des véhicules (images incluses dans le projet) apparaissent aléatoirement sur la route. Évite-les pour ne pas perdre.

    ⏸️ Pause : Appuie sur la barre d'espace pour mettre le jeu en pause.

    🌸 Récompenses : Tous les 20 véhicules évités, des fleurs apparaissent pour te féliciter.

    🏆 Meilleurs scores : Les scores sont enregistrés dans la base de données racing_car_game et affichés en temps réel.

    💥 Explosions : En cas de collision, une animation d'explosion est déclenchée.

    🎮 Boutons interactifs : Des boutons pour redémarrer, quitter le jeu ou afficher les instructions.

Prérequis

    Serveur local : Utilise XAMPP ou WAMP pour exécuter le projet.

    Base de données : Importe la base de données racing_car_game incluse dans le projet.

    Navigateur web : Ouvre index.php dans ton navigateur pour jouer.(je l'ai nommé chez moi indice.php parce que ce n'est pas le projet initial)

Structure du Projet

    index.php : Fichier principal du jeu.

    Base de données : racing_car_game pour enregistrer les scores.

    Images : Photos des véhicules utilisées comme obstacles.

Comment utiliser le projet ?

    Télécharge le code ou clone ce dépôt.

    Place le projet dans le dossier htdocs (XAMPP) ou www (WAMP).

    Importe la base de données racing_car_game dans phpMyAdmin.

    Ouvre index.php dans ton navigateur.

    Amuse-toi à battre ton meilleur score !
